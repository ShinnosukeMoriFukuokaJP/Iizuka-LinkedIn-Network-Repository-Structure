Fukuoka Companies Database

A comprehensive database of registered companies operating in Fukuoka Prefecture, Japan. This dataset contains detailed information about businesses across various industries, including contact details, business classifications, and operational status.

Purpose

Facilitate business intelligence, market research, and B2B networking for companies operating in the Fukuoka region. Enables identification of potential partners, competitors, and market opportunities.

Sample Fields

Field Name Data Type Description
company_id string Unique identifier for each company
company_name string Official registered company name
industry string Primary industry classification (JISC standard)
establishment_date date Company founding or registration date
employee_count integer Number of employees (estimated)
headquarters_address string Full address of headquarters location
phone_number string Primary business contact number
website_url url Company official website
revenue_usd decimal Annual revenue in USD (when available)
status enum Operating status: active, inactive, dissolved
last_updated timestamp Last record update timestamp

Job Listings Dataset

An active collection of employment opportunities across Fukuoka and surrounding regions. This dataset aggregates job postings from multiple sources, providing detailed information about positions, requirements, and compensation across various sectors.

Purpose

Support job seekers, career analysts, and recruiters with comprehensive labor market intelligence. Enable trend analysis, skills gap identification, and compensation benchmarking across industries and experience levels.

Sample Fields

Field Name Data Type Description
job_id string Unique identifier for each job listing
job_title string Position title or job name
company_id string Reference to employer in Companies Database
job_category string Functional category: IT, Sales, Engineering, HR, etc.
employment_type enum Full-time, Part-time, Contract, Temporary
experience_level enum Entry-level, Mid-level, Senior, Executive
salary_min_jpy integer Minimum annual salary in JPY
salary_max_jpy integer Maximum annual salary in JPY
job_location string City and prefecture of job location
required_skills array List of required technical or professional skills
job_description text Detailed job description and responsibilities
posted_date timestamp Date job listing was posted
deadline date Application deadline (if specified)
application_url url Link to apply for the position
