## 使い方 / How to Use

### このリポジトリを自分の旅行に活用する

```bash
# 1. Fork する（GitHub の画面右上）

# 2. Clone する
git clone https://github.com/[your-username]/[repo-name]

# 3. テンプレートをカスタマイズ
# templates/travel-record.md を自分の旅行用に編集

# 4. 記録して push
git add .
git commit -m "YYYY-MM-DD: [都市名] Day1記録追加"
git push
```

### iPhone だけで運用する場合

1. GitHub アプリ または Working Copy を使用
2. iPhone のメモアプリで下書き
3. AI ツール（Claude等）で整形
4. GitHub に push
