## 投稿チェックリスト

### 投稿前
- [ ] リポジトリが Public になっているか確認
- [ ] GitHub Pages が表示されるか確認
- [ ] `[  ]` の置換漏れがないか確認
- [ ] URLが正しいか確認

### 投稿時
- [ ] 英語版を投稿（メイン）
- [ ] 日本語版を投稿（翌日 or 同日）
- [ ] リポジトリ URL を必ず含める
- [ ] ハッシュタグは最大10個まで

### 投稿後
- [ ] エンゲージメントをモニタリング（24時間後）
- [ ] コメントへの返信
- [ ] Forkされたか確認
