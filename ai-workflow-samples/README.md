# AI Workflow Samples

> **Problem**: Most professionals say they "use AI" but cannot show how.
> **Solution**: Three documented, reproducible AI workflows — meeting automation, task management, and email parsing.
> **Impact**: Verifiable GitHub evidence that converts a resume claim into a portfolio signal.

---

This repository demonstrates practical AI-based workflow improvements for business and personal productivity.

## Overview

Practical use cases of AI (ChatGPT), Notion, and Python automation to improve daily workflows.
Each workflow is fully documented and reproducible.

## Contents

### 1. ChatGPT Use Case — Meeting Summary Automation
- Raw notes → structured minutes in one prompt
- Outputs: decisions, issues, action items, next steps

### 2. Notion Template — Task Management System
- Daily tasks, weekly review, project tracking
- Designed for solo operators and small teams

### 3. Python Automation — Email Summarization Concept
- Extracts deadlines, action items, and meeting flags
- Conceptual proof ready to connect to any real API

## Tools Used

- ChatGPT
- Notion
- Python

## Purpose

Convert the sentence "I use AI" into verifiable, linked evidence.
