# Simple Task Management System

> **Problem**: Solo operators lose track of priorities and weekly progress without a structured system.
> **Solution**: A Notion-based task management template with daily tasks, weekly reviews, and project tracking.
> **Impact**: Consistent weekly review rhythm that reduces dropped tasks and improves focus.

---

## Structure

### Daily Tasks
| Field | Options |
|---|---|
| Task Name | Free text |
| Priority | High / Medium / Low |
| Status | Not Started / In Progress / Done |

### Weekly Review (Every Friday)
- What was completed?
- What is delayed?
- What should be improved next week?

### Project List
| Field | Description |
|---|---|
| Project Name | Short title |
| Objective | One-line goal |
| Deadline | Target date |
| Current Status | On track / At risk / Blocked |

## Workflow

1. Add tasks daily (morning, 5 minutes)
2. Review progress every Friday
3. Adjust priorities weekly based on review output
