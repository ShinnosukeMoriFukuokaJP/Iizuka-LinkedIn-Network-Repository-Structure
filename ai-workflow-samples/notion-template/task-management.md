# Simple Task Management System (Notion Style)

## Structure

### Daily Tasks
- Task Name
- Priority (High / Medium / Low)
- Status (Not Started / In Progress / Done)

### Weekly Review
- What was completed?
- What is delayed?
- What should be improved?

### Project List
- Project Name
- Objective
- Deadline
- Current Status

## Workflow

1. Add tasks daily
2. Review progress every Friday
3. Adjust priorities weekly
