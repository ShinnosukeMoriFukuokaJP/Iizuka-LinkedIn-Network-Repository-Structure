# iizuka-travel-intel

**飯塚市の現地データを集積するオープンインテリジェンスリポジトリ**

> JLI-OS の Iizuka City Node に、旅行者・訪問者視点の実地データを追加するプロジェクト。
> 「地元民が作るローカルガイド」として差別化された知識インフラ。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://shinnosukemorifukuokajp.github.io/iizuka-travel-intel/)

---

## 概要

飯塚市（福岡県・筑豊地域）に関する以下の情報を構造化して公開します。

- アクセス情報（交通・費用・所要時間）
- 地域産業・SME動向
- 観光・グルメスポット
- デジタル化・DX事例
- Indo-Pacific 文脈での位置づけ

---

## 貢献方法

1. Fork する
2. 情報を追加・修正する
3. Pull Request を送る

> ⚠️ 個人情報（氏名・連絡先等）を含むデータは提出しないでください。
> 事業所情報は公開情報のみ記載してください。

---

## 免責事項

- 掲載情報は訪問・調査時点のものです。最新情報は各公式サイトでご確認ください。
- 第三者の個人情報・肖像は掲載していません。
- 掲載サービス名は各社の商標です。本リポジトリは各社と無関係です。
