## Internal Linking Rules

### Every page must have:
1. **Breadcrumb** at top: `[← Parent Section](../README.md)`
2. **Footer** at bottom: `*Iizuka LinkedIn Network · [Context] · Chikuho Region · Fukuoka Prefecture · Japan*`
3. **At least 3 internal links** to related pages

### Cross-linking priorities:
- City pages → Companies in that city → Jobs in that city → Events in that city
- AI resources → AI companies → AI jobs → AI events
- Success stories → Related resource pages → LinkedIn Group

## GitHub Pages Implementation

When GitHub Pages is activated:
1. Every `.md` file becomes a crawlable HTML page
2. The repository URL becomes: `https://[username].github.io/[repo-name]/`
3. Google indexes all pages within 2–4 weeks
4. Internal links pass SEO authority between pages

**To activate GitHub Pages:**
Settings → Pages → Source: main branch → Save
