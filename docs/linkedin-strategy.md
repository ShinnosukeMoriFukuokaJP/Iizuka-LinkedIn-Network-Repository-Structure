The complete LinkedIn integration and cross-platform growth strategy for ILN.

## LinkedIn Ecosystem Architecture

```
LinkedIn Group (Community Hub)
        ↓
LinkedIn Newsletter (Monthly Digest)
        ↓
LinkedIn Events (Event Promotion)
        ↓
Personal Profile Posts (Content Distribution)
        ↓
GitHub Repository (Knowledge Base)
```

Every layer reinforces the others. The GitHub repository is the brain. LinkedIn is the voice.

## LinkedIn Group Strategy

### Group Setup
- **Name**: Iizuka LinkedIn Network — Chikuho Region Professional Community
- **Type**: Public (anyone can join, posts visible to all)
- **Description**: Use all target keywords naturally
- **Rules**: Mirror the Code of Conduct, with LinkedIn-specific additions

### Content Calendar (Weekly)

| Day | Post Type | Example |
|-----|-----------|---------|
| Monday | Welcome / New member spotlight | "Welcome to [Name] from Tagawa City!" |
| Tuesday | Resource share | Link to new GitHub resource with context |
| Wednesday | Discussion question | "What's the biggest AI challenge for Chikuho SMEs?" |
| Thursday | Event promotion | Upcoming event in Iizuka or online |
| Friday | Success story or insight | Member win or regional news |

### Group Post Formats That Drive Engagement
1. **Open questions** — "What AI tool are you using in your Chikuho business?"
2. **Local insights** — "What I'm seeing in Iizuka City's business community..."
3. **Resource drops** — "New resource added to the GitHub repo: [link]"
4. **Member spotlights** — Feature an active contributor
5. **Event recaps** — Summary of what happened at the last meetup

## LinkedIn Newsletter

### Newsletter: "Chikuho Professional Digest"

**Frequency**: Monthly (last Monday of the month)  
**Target subscribers**: 200 (Month 3) → 500 (Month 6) → 1,000+ (Month 12)

### Issue Structure
```
Subject: Chikuho Professional Digest — [Month] [Year]

1. OPENING (100 words)
   One insight or observation about the Chikuho Region this month

2. AI SPOTLIGHT (200 words)
   One AI tool or use case relevant to regional businesses

3. DX CASE STUDY (200 words)
   One Digital Transformation story from Fukuoka Prefecture

4. CAREER CORNER (150 words)
   One LinkedIn tip or job market insight

5. STARTUP FEATURE (150 words)
   One founder story or funding opportunity

6. EVENTS THIS MONTH (100 words)
   Upcoming events with dates and links

7. NEW RESOURCES (100 words)
   What was added to the GitHub repo this month

8. CLOSING (50 words)
   Call to action: join, contribute, share
```

### Newsletter Growth Tactics
- Mention newsletter in every LinkedIn post
- Link from GitHub README and START_HERE.md
- Promote at events: "Sign up for our monthly digest"
- Cross-promote in partner organizations' channels
- First 3 issues: promote heavily on personal profiles

## LinkedIn Events Strategy

### For every in-person event:
1. Create a LinkedIn Event 4+ weeks in advance
2. Invite LinkedIn Group members
3. Post reminder at 2 weeks, 1 week, and day before
4. Post a recap with photos within 48 hours
5. Tag attendees who consent

### For online webinars:
1. LinkedIn Event + Zoom or Google Meet link
2. Record for later distribution
3. Post highlights as short clips (if available)
