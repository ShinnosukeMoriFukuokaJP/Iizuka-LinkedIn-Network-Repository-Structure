## Internal Linking Rules

1. Every city page links to: Companies → Jobs → Events → Resources
2. Every resource page links to: Related city pages + Related jobs
3. Every job listing links to: Company page + City page + Career resources
4. README.md links to all top-level sections
5. Every page has `[← Parent](parent/README.md)` breadcrumb
