# travel-oss-template

**旅行プロジェクトをGitHub OSSとして公開するためのスターターキット**

> Fork して自分の旅行に使ってください。iPhone + 無料プランのみで完結します。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## このテンプレートでできること

- 旅行記録を構造化された Markdown として残す
- GitHub Pages で旅行記を公開
- JLI-OS 互換の都市データを作成
- LinkedIn 投稿を英語・日本語で準備
- iPhone のみ・無料プランで全工程完結

---

## クイックスタート

```
1. 右上の「Use this template」をクリック
2. リポジトリ名を決める（例: kyoto-trip-2026）
3. CHECKLIST.md に従って進める
```

---

## ディレクトリ構成

```
travel-oss-template/
├── README.md
├── CHECKLIST.md
├── travel-record.md
├── city-node.md

    ├── linkedin-post-en.md

```

- 掲載サービス名（GitHub、LinkedIn 等）は各社の商標です。本テンプレートは各社と無関係です。
- 旅行記録の公開時は第三者の個人情報・肖像権に配慮してください（個人情報保護法）。
- 各サービスの無料プランは変更される場合があります（2026年6月時点を前提）。
