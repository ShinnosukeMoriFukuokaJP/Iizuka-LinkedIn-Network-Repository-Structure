# Markdown Formatter Prompt

## 用途
旅行中にiPhoneで書いた箇条書きメモを、GitHub公開用Markdownに整形する。
旅行中・旅行後に使用。

---

## プロンプト本文（コピペ用）

```
以下の旅行メモをGitHub公開用のMarkdown記事に整形してください。

# 条件
・対象読者：GitHubユーザー・OSS開発者・旅行好き
・形式：見出し・箇条書き・表を使ったReadable Markdown
・品質：READMEとしてそのまま公開できるレベル
・個人情報注意：他者の氏名・顔写真情報は含めないでください

# 出力形式
## [日付] — [場所名]
### 移動
### スポット
### 気づき

# メモ
[ここにiPhoneで書いたメモを貼り付ける]
```

---

## 入力例

```
6/27 長崎着 新幹線 2時間 3000円
グラバー園 坂道 きつい でも景色良い 500円
ちゃんぽん食べた 麺屋かわ 600円 おいしい
出島 復元されてる 面白い 510円
夜 中華街
```

## 出力例（抜粋）

```markdown
## 2026-06-27 — 長崎市

### 移動
- 交通手段：新幹線
- 所要時間：約2時間
- 費用：¥3,000

### スポット

#### グラバー園
- 費用：¥500
- 評価：⭐⭐⭐⭐
- メモ：坂道が急だが、長崎港を見渡す景色は圧巻。

#### 出島
- 費用：¥510
- メモ：江戸時代の復元建築。歴史資料として価値が高い。
```

## 注意事項

- 店名・施設名は公開情報として記載可能
- 他者の氏名・個人情報は整形後も含めないこと
- 価格・時間等の数字は確認してから公開すること
