# GitHub Pages Configuration

To enable GitHub Pages for this repository:

1. Rename `_config/jekyll-config.yml` to `_config.yml` and move it to the repository root
2. Go to **Settings → Pages** in the GitHub repository
3. Set source to **main branch**
4. GitHub Pages will automatically build and publish the site

## Custom Domain (Optional)

To use a custom domain (e.g., `iizuka-linkedin-network.org`):
1. Add a `CNAME` file to the repository root with your domain
2. Configure DNS with your domain registrar
