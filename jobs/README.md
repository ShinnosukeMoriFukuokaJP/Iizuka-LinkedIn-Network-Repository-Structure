## Post a Job

Submit a listing via Pull Request using the format below:

```markdown
### [Job Title] — [Company Name]
**Category:** Remote / AI / DX / Local / Internship / Volunteer
**Location:** City or Remote
**Type:** Full-time / Part-time / Contract / Freelance
**Salary:** ¥X,XXX,XXX ~ / Negotiable / Volunteer
**Posted:** YYYY-MM-DD
**Apply by:** YYYY-MM-DD or Open
**Description:** What the role involves.
**Requirements:** Key skills needed.
**Apply:** [URL or email]
```
